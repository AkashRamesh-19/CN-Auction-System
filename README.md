# Secure Multi-Client Auction System

## Description
A secure auction system using TCP sockets and SSL.

## Features
- Multi-client support
- Secure communication (SSL)
- Real-time bidding
- Load testing

## How to Run
1. cd certs
2. python generate_cert.py
3. cd ..
4. python server.py
5. python client.py
6. python load_test.py
